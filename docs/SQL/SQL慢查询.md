SQL慢查询




影响sql执行速度的因素
1、网络，各个节点之间的网络
2、OS系统 ，即数据库服务器
3、MySQL数据库本身


网络丢包，重传
网卡满 比如大字段
网络链路变长：多个节点调用

磁盘io被其他任务占用

4.1 没有索引，或者索引不正确
4.2 隐式转换：字段类型不一样、in 参数包含多个类型、联表查询时候两表字符集类型不一致
4.3 执行计划错误：由于MySQL优化器本身的不足，选择执行计划时会导致错误的执行计划使sql走了错误的索引或者没有做索引。
4.4 数据巨大
4.5 MetaData Lock锁等待
4.6 并发更新同一行
